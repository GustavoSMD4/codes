🔐 DB com senhas de 4 dígitos (0-9) - API

API para análise e estatísticas de códigos numéricos de 4 dígitos.

🚀 Endpoints

📊 Estatísticas gerais

GET /stats

Retorna estatísticas gerais do banco:

  • 🔢 Total de senhas
  • 🥇 Código mais usado
  • 🥄 Código menos usado

Exemplo:

https://gustavosmd4codes.pythonanywhere.com/stats

📄 Listar todos os códigos

GET /codes

https://gustavosmd4codes.pythonanywhere.com/codes

🥇 Listar primeiros códigos

GET /codes/first/<quantidade>

https://gustavosmd4codes.pythonanywhere.com/codes/first/10

🏁 Listar últimos códigos

GET /codes/last/<quantidade>

https://gustavosmd4codes.pythonanywhere.com/codes/last/10

🔍 Verificar posição de um código

GET /code/verify/<code>

https://gustavosmd4codes.pythonanywhere.com/code/verify/1234

🏆 Sugerir código seguro ou comum

GET /code/suggest?safe=<True/False>

Parâmetros:

  • safe=True → código menos comum (mais seguro)
  • safe=False → código mais comum
https://gustavosmd4codes.pythonanywhere.com/code/suggest?safe=True

📊 Gerar planilha Excel por prefixo

GET /codes/planilha/<digits>

  • digits aceita valores: 1, 2 ou 3
  • Retorna prefixos únicos
https://gustavosmd4codes.pythonanywhere.com/codes/planilha/3

📄 CSV completo

GET /codes/csv

Retorna download automático do CSV contendo todos os códigos.

https://gustavosmd4codes.pythonanywhere.com/codes/csv

🧠 Heatmap de dígitos por posição

GET /codes/stats/<limit>

Retorna visualização HTML com:

  • 🔥 Heatmap de frequência
  • 📈 Porcentagens por posição
  • 🥇 Dígito mais frequente
  • 📊 Estatísticas gerais
https://gustavosmd4codes.pythonanywhere.com/codes/stats/1000

🔥 Heatmap por intervalo

GET /codes/stats/between/<start>/<end>

  • start → código inicial
  • end → código final

Regras:

  • start deve ser ≤ end
  • caso contrário retorna erro 400
https://gustavosmd4codes.pythonanywhere.com/codes/stats/between/1000/2000
